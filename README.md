ASCIIStone is a 2D multiplayer survival game with console ASCII graphics.
The game has random terrain generation, enemies with pathfinding, lighting

WASD to move
Arrow keys to attack/build/use item
E for inventory
C for crafting
Esc for escaping
Shift for realizing there's no sprint

Want to make a screenshot?
It's a console, just copy the whole screen into a text file and you have it :D
